- Restful API menggunakan Laravel Passport
login to get token : 
(post) http://127.0.0.1:8000/api/v1/login
request : email & password

create article : 
(post) http://127.0.0.1:8000/api/v1/createArticle
request : title, content, image, and category_id

list all :
(get) http://127.0.0.1:8000/api/v1/listAll

show detail :
(get) http://127.0.0.1:8000/api/v1/showDetail

update article :
(post) http://127.0.0.1:8000/api/v1/updateArticle
request : id, title, content, image, category_id
 
delete :
(post) http://127.0.0.1:8000/api/v1/deleteArticle
request : id
